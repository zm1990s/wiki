# 此处收藏 powercli 脚本

powercli 是 VMware 出品的一系列 powershell 模块，通过这些模块，用于可以非常快捷的利用 powershell 对vCenter 及其相关资源进行批量操作，例如批量修改 ESXI 主机的配置、列举虚拟机清单等等。



[脚本收藏包](https://wiki.halfcoffee.com/docs/powerclipack.zip)
