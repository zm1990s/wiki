# 查看操作系统版本等信息

## 目录
{: .no_toc .text-delta }

1. TOC
{:toc}

## Ubuntu 

### 查看版本

```shell
$ lsb_release -a
No LSB modules are available.
Distributor ID: Ubuntu
Description:    Ubuntu 14.04.3 LTS
Release:        14.04
Codename:       trusty
```

### 查看内核

```shell
uname -a
```





## Centos 

### 查看版本

```shell
cat /etc/redhat-release
```

### 查看内核

```shell
uname -a
```
