# Azure AD

如果用户本地没有 Windows AD，且不想费心搭建 OpenLDAP，Azure AD 则是个很好的选择。



## LDAPS

默认 LDAP 使用非安全的协议，在 Azure AD 上可以开启 SSL/TLS 功能，使用安全的 LDAPS。

配置步骤：

- 为 Azure AD 创建数字证书
- 启用 Secure LDAP
- 配置 internet 可以访问 Secure LDAP
