# tcpdump 抓包

个人比较喜欢用wireshark，所以希望可以用tcpdump来抓出完整的数据包，命令如下：

```shell
tcpdump -i bond0 -s 0 -w test.pcap
# -s 0 时表示将数据包的大小设置最大，一般是65535
# -i 表示需要抓包的接口
# -w 表示输出成文件，后面加文件名
```

抓指定端口/类型的包

```shell
 tcpdump -i any -nnnvvvSs0 port 80 or icmp -w test.pcap
 # 抓取所有接口的 HTTP 报文和 ICMP 报文
```
