## 介绍

做 DoH 测试时发现给浏览器配置了似乎没生效，找了下可以用 curl 来进行简单的测试验证连通性，下面记录下过程：

### Cloudflare DoH 服务

该服务器国内需要🪜才能访问：

https://doh.pub/dns-query

Curl 测试：

```shell
curl --location 'https://doh.pub/dns-query?name=www.halfcoffee.com&type=A' \
--header 'accept: application/dns-json'
```

结果：

```shell
{"Status":0,"TC":false,"RD":true,"RA":true,"AD":false,"CD":false,"Question":[{"name":"www.halfcoffee.com.","type":1}],"Answer":[{"name":"www.halfcoffee.com.","type":1,"TTL":300,"data":"104.21.66.176"},{"name":"www.halfcoffee.com.","type":1,"TTL":300,"data":"172.67.206.69"}]}%
```
